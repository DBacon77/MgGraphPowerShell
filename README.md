# MgGraphPowerShell
Scripts for MgGraph PowerShell to manage Azure AD.

LimitGroupCreate - Microsoft Graph (beta) Powershell to limit group creation in M365. Taking the prior AzureAD Powershell for limiting group creation: (https://learn.microsoft.com/en-us/microsoft-365/solutions/manage-creation-of-groups?view=o365-worldwide) and changing it to work with Microsoft Graph Powershell.

UserGroups - List all users and the groups they are members of.
